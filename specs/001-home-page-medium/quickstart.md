# Quickstart: Home Page Inspirada no Medium

## Pre-requisitos

- Node.js >= 20
- Dependencias instaladas

## Executar localmente

```bash
cd /home/ramiro/Documentos/Projetos/hub-ti/hub-ti
npm install
npm run dev
```

Abra http://localhost:3000 para visualizar a home.

## Dados de conteudo

- Os conteudos sao lidos de /home/ramiro/Documentos/Projetos/hub-ti/hub-ti/content
- Imagens devem estar em /home/ramiro/Documentos/Projetos/hub-ti/hub-ti/public

## Validacao rapida

- Verifique se a home apresenta sidebar, feed central e coluna direita.
- Confirme que cards exibem titulo e resumo, e que topicos aparecem na coluna direita.
